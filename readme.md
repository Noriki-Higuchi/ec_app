# ECアプリケーション

## 概要
LaravelでECサイトを作成しました。

## 目的
テーブルのジョイン、検索機能、ページネーション、セッション機能を実装できるようになること。ややロジックの絡むWebアプリケーションの作成に慣れること。

## 画面別機能一覧
* 全画面共通
    * ヘッダーに検索バーがあり、商品を検索できる
* ホーム画面(/)
    * 商品一覧を閲覧できる
    * 商品はページネーション機能を使う
    * ログインユーザーのみカートに商品を入れられる
* カート一覧画面(/cartitem)
    * 自分がカートに入れた商品と個数、小計を表示する
    * カートの商品の個数変更、削除ができる
    * 購入画面へ遷移するボタンを配置する
* 購入画面(/buy)
    * お届け先入力フォームを表示する
    * カート内容を表示する
    * 入力内容を確認ボタンを押すと、同じBladeで入力内容を表示し、確定か入力内容変更を選ぶ
    * 確定すれば購入完了画面を表示してユーザーのカート情報を削除する、変更の場合はフォームを表示する
